#  Prototype 프로퍼티


상속을 구현하는 용도의 프로토타입 객체다.
생성자함수로 호출할 수 있는 함수객체이다.

모든 객체는 `[[Prototype]]`이라는 내부 슬롯을 갖는데,
함수가 생성자 함수로 호출될 때 생성자 함수가 생성할 인스턴스의 프로토타입 객체를 가리킨다.
