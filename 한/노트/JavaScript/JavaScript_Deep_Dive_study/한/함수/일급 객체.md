# 일급 객체
>js의 함수는 객체 타입의 값이기 때문에 **일급객체**다
>이는 함수를 값처럼 자유롭게 사용할 수 있다는것을 의미한다.

- 익명 함수로 생성 가능.
- 변수나 자료구조(객체,배열)에 할당 가능.
- 함수의 매개변수에 전달 가능.
- 함수의 반환값으로 사용 가능.
