1. 변수 선언이란?

  • 변수를 생성하는 것
  • 변수를 사용하려면 반드시 선언이 필요
  • 변수를 선언하는 키워드로는 var, let, const 키워드를 사용
ES5 vs ES6
  • ES5
    • var 키워드
  • ES6
    • let, const 키워드
    • 기본적으로 하위 호환성 유지(ES5지원)
    • ES6는 ES5의 상위 집합
키워드
  • 명령어
  • 자바스크립트 엔진은 키워드를 만나면 자신이 수행해야 할 약속된 동작을 수행
undefined
  • 자바스크립트에서 제공하는 원시타입의 값
  • 변수에 값이 할당되지 않음

2. 변수 선언 단계

  1. 선언 단계 : 변수 이름을 등록
  2. 초기화 단계 : 암묵적으로 undefined를 할당해 초기화
변수 이름은 어디에 등록?
  • 실행 컨텍스트
  • 변수 이름과 변수 값은 key - value 형식인 객체로 등록되어 관리
참조에러(ReferencError)
  • 선언하지 않은 식별자에 접근하면 발생
